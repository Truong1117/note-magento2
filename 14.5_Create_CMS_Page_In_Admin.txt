Assign a Home Page 

You can assign any page as a Home Page according to the following guide:

~ Finish all steps to create a new page.
~ On the Admin Panel, Stores > Settings > Configuration.
~ On the left panel, under General, select Web.
~ Open the Default Pages section,
	~~ Set CMS Page as Home Page in the CMS Page field.
	~~ Save Config to activate the settings.
~ Click on the Cache Management link in the system message to flush your cache.
