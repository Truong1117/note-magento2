Using jQuery UI styles

link: https://devdocs.magento.com/guides/v2.4/frontend-dev-guide/css-topics/css-jquery.html

I. jQuery UI styles used on the storefront

~ In Magento out of the box certain storefront elements are implemented using jQuery UI widgets 
(like navigation menu, mini shopping cart and so on), with the default jQuery classes assigned.

~ But Magento does not contain jQuery UI styles in the code base. Styles for the certain default 
jQuery UI classes are defined in Magento by default (in the default Magento storefront and Admin themes). 
For other jQuery UI classes styles are not defined.
